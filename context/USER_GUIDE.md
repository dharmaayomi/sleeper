# macOS Porto - User & Feature Guide

## 👋 Welcome to macOS Porto!

**macOS Porto** is an interactive portfolio that works like a real macOS desktop. Instead of scrolling through a traditional website, you interact with windows, a dock, and folders just like you would on a Mac computer.

---

## 🎯 What Is This?

It's a **portfolio website** — but designed as an interactive **macOS desktop**. You can:

- ✅ Drag windows around the screen
- ✅ Open and close applications
- ✅ Minimize windows to the dock
- ✅ Explore projects in folders
- ✅ View tech skills and blog posts
- ✅ Get contact information
- ✅ Download resume as PDF

---

## 🚀 Getting Started

### On Desktop (Best Experience)

1. Open the website in a web browser
2. You'll see a macOS desktop with a dock at the bottom
3. Click any app in the dock or navigation bar at the top

### On Mobile

- The experience is optimized for desktop
- You'll see a notification suggesting desktop view
- Responsive layout available, but some features may be limited

## 🪟 Understanding Windows

### What Are Windows?

Windows are like **applications** on a Mac. Each window shows different content:

| Window       | What It Shows                 | Icon          |
| ------------ | ----------------------------- | ------------- |
| **Finder**   | Projects and portfolio items  | Folder icon   |
| **Safari**   | Blog posts and articles       | Compass icon  |
| **Terminal** | Tech skills and expertise     | Console icon  |
| **Resume**   | PDF of resume (downloadable)  | Document icon |
| **Contact**  | Contact form and social links | Chat icon     |
| **Photos**   | Photo gallery                 | Photo icon    |
| **Text**     | Notes and text editor         | Document icon |
| **Image**    | Image viewer                  | Picture icon  |

---

## 🎮 How to Use Windows

### Opening a Window

**Method 1: Click the Dock**

- Look at the bottom of the screen
- Click any app icon to open it
- A window pops up on the desktop

**Method 2: Use Navigation Bar**

- Look at the top menu bar
- Click "Projects", "Contact", or "Resume"
- Opens the corresponding window

### Window Controls (Top-Right Corner)

Each window has three colored buttons:

| Button       | Color  | Action                      |
| ------------ | ------ | --------------------------- |
| **Close**    | Red    | Closes the window           |
| **Minimize** | Yellow | Hides window, moves to dock |
| **Maximize** | Green  | (Currently for decoration)  |

### Moving Windows

- Click and drag the window by its title bar
- Move it anywhere on the screen
- Window stays where you put it

### Closing Windows

- Click the **red close button** (⊗)
- Or click the dock icon again while window is open
- Window disappears and app closes

### Minimizing Windows

- Click the **yellow minimize button** (−)
- Window shrinks and moves to the dock
- Click dock icon again to bring it back

## 🎯 The Dock

### What Is It?

The dock is the bar at the bottom of the screen showing all available apps.

### Features

- **Hover over icons** — Icons scale up as you hover
- **Click to launch** — Opens the app
- **Visual indicator** — White dot = app is open; Gray dot = minimized
- **Disabled apps** — Some apps appear grayed out (archive, trash)

### App Icons in Dock

1. **Portfolio** (Finder) — Browse projects
2. **Articles** (Safari) — Read blog posts
3. **Gallery** (Photos) — View photo gallery
4. **Contact** — Get in touch
5. **Skills** (Terminal) — Tech skills
6. **Archive** (Trash) — Archived items

## 📱 Mobile Experience

### Desktop Recommended

- Dock is hidden on small screens
- Windows may overlap awkwardly
- Touch interactions limited
- Warning message shown on mobile

### What You Can Still Do

- Open windows by tapping navigation links
- Scroll through window content
- View all information, just less polished

### Best Practice

- Use on desktop/laptop for full experience
- Large screen for dragging windows comfortably
- High resolution displays look better

## 🆘 Troubleshooting

### Window Not Opening?

- Click dock icon again
- Try navigation link instead
- Refresh page if stuck

### Animation Seems Slow?

- Close other browser tabs
- Disable browser extensions
- Try different browser
- Usually smooth on modern hardware

### Mobile Experience Issues?

- Use desktop browser instead
- Landscape orientation sometimes better
- Full experience on computer recommended

### Download Not Working?

- Try different browser
- Check downloads folder
- May be browser-specific
